## Requirements
Globals: `npm`, `ng`, `tsc`

## Build Frontend
`cd frontend`
`npm install`
`ng build`

## Build Backend
`cd server`
`npm install`
`npm run-script build`

## Start server
`npm install`
`npm run-script start`
